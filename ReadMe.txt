To run the notebooks, you have to go in this order :
- Data Preparation
- Feature Engineering
- Models Training
- Predictions
- Shap

To be able to use the mlflow ui, you have to open your terminal, go to the directory of your project and execute "mlflow ui".
It will show you a local address where all the runs are saved.

To open the sphinx website, you have to go into docs/build/html and open a html file (by default, open index.html)
